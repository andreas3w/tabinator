Tabs matching these URL patterns are closed automatically when the tab limit is hit, before prompting you. Use domains like github.com or wildcard domains like *.awsapps.com. You can also match host + port like 127.0.0.1:8020.

Tabs matching these patterns are pinned on open. When Only one is enabled, opening a new matching tab replaces existing tabs with the same hostname. The first 5 patterns become quick slots for Ctrl+1 to Ctrl+5.
